Creating the Shockwave version of the movie

Shockwave movies are Director movies that are compressed for Internet delivery. Shockwave movies can be played by any browser that has the Shockwave plug-in installed.

To convert your movie to a Shockwave movie:

1 Choose File > Save As and change the file name to mychat.dir.
2 Click Save.
3 Choose File > Save As Shockwave Movie.
4 Select the Generate HTML and Preview in Browser options.
5 Click Save.
Director saves the movie as a Shockwave movie named mychat.dcr, makes a Web page for the movie called mychat.htm, and opens the movie in your browser.
