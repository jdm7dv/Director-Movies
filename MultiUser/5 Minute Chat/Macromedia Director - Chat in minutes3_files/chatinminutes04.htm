Adding behaviors to the movie

Many of Director's interactive features are preprogrammed in easy-to-use drag-and-drop behaviors. This project uses four Multiuser behaviors to add multiuser capabilities to the movie. You need to work with only the Stage and Library windows to add the behaviors and set their parameters.

To add behaviors to the movie:

1 Click the Library List pop-up menu in the upper left corner of the Library window and choose Internet > Multiuser.
2 Drag the Connect to Server behavior icon onto the Connect button (the button with the green triangle) on the Stage.
3 In the dialog box that appears, set the following options:
For Which member holds the user name? select username.
For Which member holds the user's password? select no password required.
For Which marker to go to while connecting? select loop.
For Which marker to go to when connected? select loop.
For Which marker to go to when connection fails? select loop.
For Server address enter the server IP address that you recorded from the Server Application window.
For Server port number enter 1626.
For How frequently should info be updated (in msecs)? enter 3000.
For Unique movie identification string enter mychat.dir.
For Server identification string enter server1.
4 Click OK.
5 Drag the Disconnect From Server behavior icon onto the Disconnect button (the button with the red X) on the Stage.
The default parameters that appear in the dialog box are correct.
6 Click OK.
7 Drag the Chat Input behavior onto the Enter Chat Here field at the bottom of the Stage.
The default parameters that appear in the dialog box are correct.
8 Click OK.
9 Drag the Chat Output behavior onto the light blue box that contains the text Chat Appears Here.
The default parameters that appear in the dialog box are correct.
10 Click OK.
