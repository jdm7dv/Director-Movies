Using the Shockwave Multiuser Server

Before you set up the Director movie, launch the server application that comes with Director. This server hosts the chat. When the server is running, use it to determine your computer's IP address.

If you don't have version 2.0 of the Shockwave Multiuser server, download the Shockwave Multiuser Server 2.0 updater.

To launch the server:

Double-click the Multiuser Server that comes with Director.

To determine the server's IP address:

1 Choose Status > Server to display information about the server application.
2 Write down the server IP address or copy it to the clipboard.
