Chatting

You can now have a chat on your computer Although the server and browser are on the same computer for this project, the server and chat room participants can be anywhere on the Internet.

You can now quit Director. To chat all you need to have running are the Server Application and an open browser.

To see your chat movie working:

1 Click Username in the chat window in the browser. Change it to whatever you like.
2 Click the Connect button (the button with the green triangle) in the chat movie. Once you connect to the server, you can type a message in the white box at the bottom of the window and press Enter.
Your message appears in the chat window.
To set up a second chat participant:

1 Open another browser and open the page mychat.htm again, so that the same page is open in two browsers.
2 Click Username in the new movie and change it to something other than the user name you used in the other browser.
3 Click the Connect button to connect to the Server.
4 Type a message in the white box at the bottom of the window.
Your message appears in the chat window in both browsers. Switch between browser windows to chat. You can also open mychat.htm a third time and simulate three people chatting.
If you have access to a Web server, you can place the HTML file and Shockwave movie on the server. Then you can run your chat from any Internet location. The Shockwave Multiuser Server Application must be running at the IP address you specified when making the movies, because this server hosts the chat and the chat must be running for the chat to work.
