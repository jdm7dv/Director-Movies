1. Product Information

	Runs in 15, 16, 24 and 32 bpp screen modes.
	Control for 'Duration' and 'Use Entire Stage/Change Area Only'.

        This means you can use freely redistributable versions
        as long as you want without registration,
        but use of the registered version requires purchasing it.

        * authoring mode.                                              *
	****************************************************************

	Please contact us for more information.

2. System Requirements

	DmXtremePack® is compatible with Windows 95®, Windows 98® and Windows NT4®,
        16 MB of RAM and a Pentium CPU (An MMX processor are recommended for 
	good performance). 

        DmXtremePack® is compatible with Mac OS and Power Mac

	For updated information, please consult our web site at
        http://www.dmtools.com

3. Setup

	Installation is easy. Just copy the file 
        DmXtremePack.x32 into your Director Xtras folder, and the use it 
	as all other built-in transition.
