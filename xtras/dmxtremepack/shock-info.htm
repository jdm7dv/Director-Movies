DMTools Xtras & Shockwave 7

 

Below you find the steps to use DMTools xtras with Shockwave 7. Generic information can be found on the Macromedia web-site at http://www.macromedia.com/support/director/ts/documents/sw7_xtras.htm

 

1. Modify the xtrainfo.txt file in the Director 7 folder , adding to the end of the file these lines :

;

; DMTools Xtras by Dedalomedia Interactive 

; http://www.dmtools.com/

[#namePPC:"DMPack1", #nameW32:"DMPack1.x32",#package:"http://www.dmtools.com/dmtools/shock/dmpack1"] 

[#namePPC:"DMPack2", #nameW32:"DMPack2.x32",#package:"http://www.dmtools.com/dmtools/shock/dmpack2"] 

[#namePPC:"DMXtremePack", #nameW32:"DMXtremePack.x32",#package:"http://www.dmtools.com/dmtools/shock/dmxtremepack"]

[#namePPC:"DMWaves", #nameW32:"DMWaves.x32",#package:"http://www.dmtools.com/dmtools/shock/dmwaves"]

[#namePPC:"DMStars", #nameW32:"DMStars.x32",#package:"http://www.dmtools.com/dmtools/shock/dmstars"]





2. Open a movie that uses DMTools xtras. Open the Modify\Movie\Xtras 

  dialog in director, and select the xtra in the list of xtras. The "Download 

  if needed" checkbox should now be enabled. (If it's not enabled, make sure you 

  typed correctly the lines above in step 2.) Click on the checkbox and director 

  will load information from the package file for each platform and add it to 

  the current movie. Save the movie in shockwave format.

3. Test that the package works. Open the movie from step 3in a browser or in a projector. If the xtra is not currently installed, the xtra package will get downloaded and a "Download Security Dialog" will be presented. If the xtra is already present, the xtra package will not be downloaded. The downloaded xtra will be installed in a subfolder of the xtras folder called "download".

4. Of course the xtras need to be registered with the reg-code (use one and only one reg-code even if you have one for PC and one for MAC).
