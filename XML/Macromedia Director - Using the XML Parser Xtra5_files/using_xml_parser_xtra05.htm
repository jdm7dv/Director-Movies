Accessing attributes

Some XML tags contain attributes with values. Use the Lingo attributeName and attributeValue properties to access the attributes of tags that have values. In the preceding XML example, the first tag nested inside the <e1> tag is called tagName and has two attributes called attr1 and attr2.

This Lingo uses the attributeName property to return the name of the first attribute of the tag called tagName, which is the first child of the tag <e1>: 

put gParserObject.child[1].child[1].attributeName[1]
-- "attr1"

This Lingo uses the attributeValue property with an integer to return the value of the first attribute of the tag called tagName: 

put gParserObject.child[1].child[1].attributeValue[1]
-- "val1"

This Lingo uses the attributeValue property with a string to return the value of the attribute named attr1: 

put gParserObject.child[1].child[1].attributeValue["attr1"]
-- "val1"

This Lingo uses the count function with the attributeName property to return the number of attributes in the first child of the tag <e1>: 

put gParserObject.child[1].child[1].attributeName.count
-- 2
